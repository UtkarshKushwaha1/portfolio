A user-friendly React web app that helps UPSC aspirants prepare for Prelims by providing a categorized collection of authentic past questions. Easily analyze exam trends, focus practice by topic and year, and study high-frequency subjects with a fast, interactive interface.

Key Features
  • Categorized repository of UPSC Prelims questions by topic and year
  • Analyze exam trends and the frequency of topics
  • Efficient practice using real questions from previous exams
  • Modern, responsive interface built with React
